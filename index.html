  2. Computer Vision Based Assistant System for the Visually Impaired (2024)

    For my final year project, I developed an assistant system for visually impaired individuals using advanced object detection with YOLOv8 and custom-trained models.     Developed a prototype IoT system to monitor the temperature and availability of Coca-Cola cans in real time. Used sensors to gather data and implemented the MQTT protocol for communication between the device and a smartphone application. The system provided real-time updates for accurate monitoring and timely alerts.

  4. RFID-Based Smart Door Control System (2022) |

    Designed and implemented an RFID-based access control system to enhance security and responsiveness. The system used RFID cards for authorized access and sent notifications to administrators in case of unauthorized attempts. Demonstrated expertise in embedded systems, security protocols, and real-time alert mechanisms.

  5. Pharmacy Management and Billing System (2022) |

    Developed a pharmacy management application aimed at efficiently managing inventory and generating purchase invoices. Streamlined inventory tracking, automated stock updates, and simplified invoice generation for better operational efficiency.
